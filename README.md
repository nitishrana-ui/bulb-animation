# 💡 Glowing Light Bulb Animation (HTML, CSS, JavaScript)

A realistic light bulb animation with electric current flow using pure HTML, CSS, and JavaScript — no images used!

---

## 🚀 Features

- 💡 Realistic glowing bulb
- ⚡ Electric current flowing through wire
- 🔴 Interactive ON/OFF button
- 🔊 Sound effect on click
- 📱 Responsive design

---

## 🛠️ Tech Stack

- HTML
- CSS (Animations, Gradients, Shadows)
- JavaScript

---

## 🎮 How It Works

- Click the button to toggle the light
- Wire shows animated current flow
- Bulb glows with flicker effect
- Sound plays on switch click
